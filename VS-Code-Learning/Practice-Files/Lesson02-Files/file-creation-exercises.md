## ✅ Exercise 6: Practice Folder Organization

**YOUR TASK:** Create folders and organize your files

**Follow these steps:**

1. Click **File → Open Folder**
2. Create a new folder called **"My-VS-Code-Practice"**
3. Open that folder in VS Code
4. Create these folders inside it:
   - **Week1**
   - **Week2**
   - **Projects**

**How to create folders in VS Code:**
- Right-click in Explorer → New Folder
- OR click the folder icon at the top of Explorer

5. Move your "about-me.txt" into the **Week1** folder
