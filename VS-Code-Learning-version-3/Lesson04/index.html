HTML Tags Explained

HTML uses tags to structure content. Tags look like this:

<tagname>Content goes here</tagname>

<h1> = Opening tag
Hello World! = Content
</h1> = Closing tag (notice the /)

Essential HTML Tags:

<h1> to <h6> = Headings (h1 biggest, h6 smallest)

<p> = Paragraph

<a> = Link

<img> = Image

<br> = Line break

<strong> = Bold text

<em> = Italic text

Your First HTML Page Structure:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Welcome!</h1>
    <p>This is my first webpage!</p>
  </body>
</html>

Breaking It Down:

<!DOCTYPE html> - Tells browser it's an HTML5 document

<html> - Container for all HTML

<head> - Information ABOUT the page (title, settings)

<title> - Shows in browser tab

<body> - Everything you SEE on the page

Creating Links:

<a href="https://www.google.com">Go to Google</a>

href = "hyperlink reference" (where to go)
Text between tags = What you click
