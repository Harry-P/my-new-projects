üöÄ What Are Extensions?

Think of VS Code as a smartphone. Out of the box, it's good. But with the right apps (extensions), it becomes AMAZING! üéâ

üéØ Why Use Extensions?

  • Save Time: Auto-format code, live preview, snippets
  • Look Better: Custom themes, icons, colors
  • Code Better: Error detection, auto-completion, debugging
  • Work Easier: Git integration, terminal tools, shortcuts

üì¶ How to Install Extensions

  1. Click the Extensions icon in the Activity Bar (or press Ctrl+Shift+X)
  2. Search for an extension by name
  3. Click the Install button
  4. Wait for it to install (usually a few seconds)
  5. Some extensions may require VS Code to reload

‚öôÔ∏è Configuring Extensions

Setting Prettier as Default Formatter:

  1. Open Settings (Ctrl+,)
  2. Search for "default formatter"
  3. Select Prettier - Code formatter
  4. Search for "format on save"
  5. Check the box to enable Format On Save

Now your code will automatically format every time you save! üéâ

üí° Pro Tips

  • Don't install too many extensions at once - start with the essentials
  • Read reviews and check ratings before installing
  • Disable extensions you don't use to keep VS Code fast
  • Update extensions regularly for bug fixes and new features
