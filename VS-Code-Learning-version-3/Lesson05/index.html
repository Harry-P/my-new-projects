🎨 What is CSS?

CSS = Cascading Style Sheets

🏠 Remember: HTML is the structure of a house
🎨 CSS is the paint, wallpaper, and decorations!
✨ CSS makes websites look AWESOME!

CSS Syntax (How to Write It):

selector {
  property: value;
  another-property: value;
}

Example:
h1 {
  color: blue;
  font-size: 36px;
}

3 Ways to Add CSS:

1. Inline CSS (inside HTML tag):

<h1 style="color: red;">Title</h1>

2. Internal CSS (in <head> section):

<head>
  <style>
    h1 { color: red; }
  </style>
</head>

3. External CSS (separate .css file) ⭐ BEST WAY!

<head>
  <link rel="stylesheet" href="style.css">
</head>

Essential CSS Properties:

Colors:
color: red; /* text color */
background-color: blue; /* background */

Text:
font-size: 20px;
font-family: Arial;
font-weight: bold;
text-align: center;

Spacing:
margin: 20px; /* space outside */
padding: 10px; /* space inside */

Borders:
border: 2px solid black;
border-radius: 10px; /* rounded corners */

Colors in CSS:

3 Ways to Define Colors:

• Names: red, blue, green, yellow

• Hex Codes: #FF0000, #00FF00, #0000FF

• RGB: rgb(255, 0, 0)
